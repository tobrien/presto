************
Overview
************

Facebook Presto is a distributed query engine designed to query large
datasets distributed over several heterogenous data sources.  That's
the textbook answer, the more "down-to-earth" answer is that Presto is
a faster Hive designed to not only understand SQL, but to translate
SQL queries in an execution plan that more resembles an execution plan
one would be familiar with in a traditional database.
